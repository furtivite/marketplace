# Marketplace Frontend

## Getting Started

1. **Install dependencies**
   ```bash
   npm ci
   ```
2. **Run locally**
   ```bash
   npm run dev
   ```
3. **Build for production**
   ```bash
   npm run build
   ```
4. **View Storybook locally**
   ```bash
   npm run storybook
   ```
